# Digi-Idle
Digivice-like pet game

### Current Features
- Create and load save
- Choosing Terriermon as a starter partner
- Get your partner to walk around in your farm
- [Debug] Save editor enabled for testing sprites and new Digimons

### Current Progress
- Digivolution system
